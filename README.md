# House Price Prediction (Linear Regression)

## 📌 Overview
This project predicts house prices using Linear Regression on tabular housing data.

---

## ⚙️ Pipeline
- Handled missing values using median
- Applied One-Hot Encoding for categorical variables
- Split data into training and testing sets (80/20)
- Trained Linear Regression model

---

## 📊 Results
- R² Score: **0.62**

---

## 🧠 Insights
- Linear Regression captures basic trends but struggles with non-linear relationships
- Housing data has feature interactions that linear models cannot fully capture
- Performance improved significantly after handling missing values and encoding
